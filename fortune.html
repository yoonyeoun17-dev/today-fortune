📱 홈 화면에 추가하는 방법
🍎 아이폰 (Safari)
1. 아이폰 Safari 브라우저로 이 페이지 열기
2. 하단 가운데 공유 버튼 (사각형+화살표↑) 탭
3. 스크롤 내려서 '홈 화면에 추가' 탭
4. 이름 확인 후 오른쪽 상단 '추가' 탭
5. 홈 화면에 🍀 아이콘이 생겨요!
🤖 갤럭시 (Chrome)
1. 갤럭시 Chrome 브라우저로 이 페이지 열기
2. 오른쪽 상단 점 세 개 메뉴(⋮) 탭
3. '홈 화면에 추가' 탭
4. 이름 확인 후 '추가' 탭
5. 홈 화면에 🍀 아이콘이 생겨요!
